O AWS Lambda é um serviço de computação serverless (sem servidor) orientado a eventos, fornecido pela Amazon Web Services (AWS). Ele permite que você execute códigos para praticamente qualquer tipo de aplicação ou serviço de backend sem provisionar ou gerenciar servidores. Você carrega seu código (funções) e a AWS cuida de toda a infraestrutura, executando-o automaticamente em resposta a eventos e escalando conforme a demanda
